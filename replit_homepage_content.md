**Date:** June 5, 2026

## Hero Section
The homepage currently features the headline "What will you build?" with the subheadline "Turn ideas into apps in minutes — no coding needed." It emphasizes that users can always make changes later and that the first prompt is free with no credit consumption. Quick-start options include Website, Mobile, Design, Slides, Animation, Data Visualization, 3D Game, Document, and Spreadsheet. Example prompts suggested are B2B project management app, Freelance client portal, and AI sales assistant.

## Meet Agent 4
This section highlights "Creativity runs on Replit" and introduces Agent 4 with four key capabilities:

### Infinite Canvas: Design Freely
A new space allowing users to explore and tweak designs visually and apply them directly to their apps.

### Parallel Agents: Move Faster
Parallel Agents run tasks together while keeping progress visible, handling auth, database, and design seamlessly.

### Multiple Artifacts: Ship Anything
Users can create mobile and web apps, landing pages, and videos in one project with shared design, enabling scaling without context switching.

### Support for Teams: Build Together
Teams can focus on planning while the Agent handles coordination and execution. Requests can be submitted in any order, and Agent 4 intelligently sequences and executes them optimally.

## Platform Capabilities
- **Agent Chat:** Describe and publish projects. The Agent writes production-ready code and evolves it autonomously.
- **Full Stack Infrastructure:** Built-in Authentication, Database, Hosting, and Monitoring with zero setup for scalable, secure apps.
- **Integrations:** Connect to AI and 100+ services including OpenAI, Stripe, and Google Workspace.
- **Enterprise Control:** Security controls including SSO/SAML, SOC 2, and admin controls to keep apps safe as they scale.
